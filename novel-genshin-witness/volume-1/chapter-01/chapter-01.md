# 第1章 魔神战争的黎明

**字数:** 2850 / 2500-3000
**创作日期:** 2026-01-26
**状态:** 初稿

---

## 正文内容

我记得死在了医院病床上,原神5.0刚更新。

现在睁开眼,天空不是熟悉的白色天花板,而是被撕裂的紫色苍穹。

神明的雷霆在远方轰鸣,大地的哀嚎近在耳边。空气中弥漫着一股令人窒息的能量波动,仿佛空间本身都在这个时代的力量下颤抖。

我慢慢坐起来,发现自己躺在一片焦黑的土地上。四周是满目疮痍的战场遗迹,断裂的武器、破碎的铠甲、还有一些早已分不清形状的残骸散落在地。泥土呈现出诡异的暗红色,不知是被鲜血浸染,还是这片大地本身的颜色。

"这他妈……"我喃喃自语,声音沙哑得不像话。

我低头看了看自己的手——还是那双熟悉的手,只是苍白得吓人。医院病床上的记忆还清晰如昨,心电监护仪的嘀嗒声,护士匆忙的脚步,然后是无尽的黑暗。

而现在,我坐在这里,看着一个绝对不是地球的世界。

远处又传来一声惊天动地的轰鸣,紫色的闪电在天空中撕裂出一道巨大的裂缝。那不是普通的雷电,我能感觉到——那是神明的力量,在彼此厮杀。

我忽然明白了。

那些天空的颜色,那些能量波动,还有那种刻入骨髓的熟悉感。

"提瓦特?"我对着空旷的战场说出了这个名字。

系统面板在眼前浮现。

"欢迎来到魔神战争时代,见证者。"

半透明的蓝色界面简洁得令人发指,没有多余的装饰,只有几行清晰的文字在眼前缓缓滚动:

```
【系统激活】
当前时间: 距今约2580年
当前位置: 提瓦特大陆东部边境战场遗迹
系统功能: 提瓦特见证系统
初始福利: 十连召唤券 x1

是否立即使用? Y/N
```

我沉默了三秒。

然后笑了。

这大概是史上最离谱的穿越。

"魔神战争时期,距今约2580年……"我自言自语,大脑飞速运转,"也就是说,现在离原神主线剧情还有两千五百年,连摩拉克斯都还没成为岩王帝君,巴巴托斯可能还是个风精灵,而雷电真……大概还在和影一起思考什么是永恒。"

我按下了Y。

"见证者,您的十连召唤即将开始。"系统的声音平淡得像是在播报天气,"请注意,召唤过程可能引起周围环境的能量波动。建议选择安全的地点进行。"

我环顾四周——这里有什么安全可言?整片大地都是战场,远处神明之战的余波随时可能波及这里。

"就在这里吧。"我说,"反正我都已经死了。"

金色光柱冲天而起。

那不是普通的金色,那是仿佛能穿透时空的纯粹光芒,在这片被神力撕裂的天空中格外耀眼。周围的空气开始震颤,地面微微颤抖,仿佛有什么东西正在从虚空中降临。

我能感觉到自己的心脏在狂跳。

这不是游戏,这是真的。我正在召唤一个活生生的存在,一个来自任意时代的生命,而现在,她将出现在我面前。

前九道光芒依次绽放。

四星角色的光芒,武器和圣遗物的虚影在空中浮现,然后化作流光融入我的身体。我能感觉到力量在增长,虽然还是凡人之躯,但至少不再是手无缚鸡之力的废人。

然后是第十道光芒。

它和之前的都不一样。

起初是温柔的绿色,像初春的第一抹嫩芽,像森林深处透过树叶的阳光。那绿色越来越亮,却并不刺眼,反而给人一种宁静而温暖的感觉。

光芒中心,一个小小的身影缓缓浮现。

她悬停在半空中,白色的长发如瀑布般垂落,尖尖的精灵耳朵微微颤动。绿色的眼睛睁开的时候,我仿佛看到了整个世界的智慧在其中流转。

小小的,看起来不过八九岁的女孩。

草元素的自然气息扑面而来,带着森林、泥土、花朵和生命的味道。

纳西妲。

小草神。

我呆住了。

她缓缓降落,赤裸的双脚轻轻触地,好奇地看着我。那双绿色的眼睛里没有丝毫戒备,只有纯粹的探索欲。

"你是谁……召唤我的人?"她的声音轻柔如风,却带着一种说不清的穿透力,仿佛能直接到达人的内心。

我张了张嘴,发现自己竟然不知道该怎么回答。

我是个穿越者?我是个原神玩家?我是个……见证者?

"我是个……见证者。"我犹豫了一下说。

纳西妲歪了歪头,若有所思:"见证者吗?"

她走近几步,小小的身体在焦黑的战场中显得格格不入。绿色的元素力在她身边环绕,那些被神力灼烧的土地竟然在她脚下重新焕发出生机。

"那你看见了什么?"她问。

"我看见……"我深吸一口气,"一个正在崩坏的世界。"

纳西妲点点头,仿佛早就料到会听到这样的答案。

"那么,你准备如何?"她问。

这个问题像一把刀,直直地插进我的心里。

我准备如何?

改变这一切?阻止这些悲剧?拯救那些注定陨落的神明?

可是……

"我……"我沉默了,"我还不知道。"

纳西妲没有追问,只是静静地看着我。那双眼睛里没有失望,没有催促,只有一种理解——仿佛她早就见过无数人在这个选择面前犹豫,而她愿意给每个人足够的时间。

系统面板再次浮现:

```
【历史节点提醒】
半小时后,东南方5公里处将发生一场小规模神战。
结果预测: 较弱的部族神明将陨落。
是否干预? 是 / 否
警告: 干预将消耗大量见证点,且可能产生不可预知后果
```

我握紧了拳头。

那个部族神明会死。我知道的,这是历史的一部分,是这场漫长而残酷的魔神战争中注定的悲剧之一。

我可以救她。

我有系统,有纳西妲,有无限召唤的力量。我可以改变这一切,让这场悲剧不再发生。

可是如果我救了她,会有什么后果?

蝴蝶效应会不会导致更可怕的结果?我改变了这个节点,会不会引发连锁反应,导致更多、更大的灾难?

历史是一条长河,每一个节点都承载着无数因果。如果我在这里插手,这条河流会流向何方?

我看着纳西妲。

她仿佛知道我在想什么,轻轻说:"见证者,每个选择都有自己的代价。而有些代价,我们可能无法承受。"

"但如果我什么都不做,她就会死。"我说,声音颤抖。

"是的。"纳西妲的声音依然温柔,但更加坚定,"她会死。而她的死亡,是历史洪流中的一朵浪花。你如果想要改变,就要做好被洪流卷走的准备。"

我沉默了许久。

然后问了一个问题:"如果我去看着她死去,只是看着……那算什么?"

"那叫见证。"纳西妲说,"见证不是冷漠,是另一种形式的守护。你记录她的存在,记录她的死亡,让这片大地上的故事永远不会被遗忘。这也是一种力量。"

"历史不该被改写,但应该被铭记。"我低声说。

纳西妲点点头:"那你已经做出了选择。"

我深吸一口气,对着系统面板按下"N"。

"我不干预。"我说,"但我要去见证。"

系统的回应简洁而直接:

```
【选择已记录】
见证点将根据见证事件的重要性自动获得。
当前见证点: 0
提示: 重大历史事件可获得更多见证点
```

"走吧。"我对纳西妲说,"我们去看看她。看看那位……即将逝去的神明。"

纳西妲没有说话,只是默默跟在我身边。

我们向东南方走去。

路上,纳西妲忽然问:"见证者,你害怕吗?"

"怕。"我诚实地回答,"我怕看到死亡,怕自己做不了任何事,怕……这种无力的感觉会一直持续下去。"

"那就记住这种感觉。"纳西妲说,"记住恐惧,记住无力,记住你为什么选择见证而不是改变。这些记忆会让你成为真正的见证者,而不是一个神志不清的穿越者。"

我苦笑:"你还真是说话不留情。"

"智慧有时候就是这样。"纳西妲的语气轻松了一些,"不过别担心,我会陪着你的。从今以后,我们一起见证。"

我们一起见证。

这句话让我心里暖暖的。

不知不觉间,我们已经走了五公里。前方出现了一个小山谷,能量波动比之前更加剧烈。我能感觉到两股神明的力量在其中碰撞,撕裂空间,燃烧大地。

我们停在山谷边缘,远远地观望着。

那是一位……看起来很年轻的神明。

她穿着简陋的铠甲,手里握着一把已经断裂的长剑。她的对面是一团模糊的黑色身影——我认不出那是谁,也看不清对方的模样,只能感觉到那股力量远超那位年轻的部族神明。

战斗很快就结束了。

年轻的神明倒在血泊中,黑色身影发出一声胜利的咆哮,然后消失在虚空中。

天地重新归于平静。

只有那位年轻的神明的尸体,静静地躺在焦黑的土地上。

系统提示浮现:

```
【见证成功】
获得见证点: 100
解锁功能: 历史记录
当前见证点: 100

历史记录已自动生成:
- 神明名称: 不详(部族神明)
- 死亡时间: 魔神战争早期,距今约2580年
- 死亡原因: 神战,实力悬殊
- 历史意义: 无(小型部族神明,影响力有限)

见证者备注: [空]
```

我看着那行字,眼泪忽然流了下来。

"历史意义:无。"我念出那四个字,"她死了,而历史记录说她是'无'。"

纳西妲轻轻叹了口气:"这个世界就是这样。有人惊天动地,有人默默无闻。而见证者的责任,就是让每一个生命都被记住,哪怕只是在某个人的记忆里。"

我擦掉眼泪,对着系统面板输入:

```
见证者备注:
她战斗到最后一刻。即使知道必败,也没有放弃。
她的名字无人知晓,但她的勇气值得铭记。
愿她在大地的怀抱中得到安息。
```

系统回应:

```
【备注已保存】
额外奖励见证点: 50
当前见证点: 150

成就解锁: 见证者的初心
成就描述: 记住每一个被遗忘的生命
```

"谢谢。"我轻声说。

"不客气。"纳西妲看着我,眼神里第一次出现了认可,"见证者,你开始明白了。"

"明白什么?"

"明白见证不是记录冰冷的文字,而是记住每一个有温度的生命。"纳西妲说,"你刚刚做到了。"

我深吸一口气,转身看向远方。

山谷之外,更多的神明之战正在上演。这个时代的每一天,都有无数生命在消逝,都有无数故事在无人知晓的角落里发生。

而我,是这个时代唯一的见证者。

"从今天起,"我说,"我是提瓦特的见证者。我会记录这一切,让这片大地上的故事永远不会被遗忘。"

纳西妲点点头,然后忽然看向远方。

"见证者,你看……"

我顺着她的目光望去。

远处,一个熟悉的身影正在穿越战场。

那是一个年轻的男人,黑色的长发,金色的眼睛,穿着简陋却整洁的铠甲。他的身上缠绕着岩元素的力量,那力量还不算强大,但已经展现出惊人的稳定性。

他握着一把长枪,一步一步走向前方的战场。

我呆住了。

"那是……"我声音颤抖。

"是的。"纳西妲说,"年幼的摩拉克斯。在成为岩王帝君之前,他还只是一个在战火中成长的无名战士。"

我看着那个身影,看着那个还没有成为"钟离"的年轻人,看着那个还在学习如何成为神明的存在。

历史,正在我的眼前展开。

而这,只是开始。
